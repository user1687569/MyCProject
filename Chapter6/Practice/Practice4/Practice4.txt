4. 下列哪条语句和其他两条语句不等价?
(a) for(i = 0; i < 10; i++) ...
(b) for(i = 0; i < 10; ++i) ...
(c) for(i = 0; i++ < 10; )  ...
答: 最后一条语句和其他两条语句不等价

5. 下列哪条语句和其他两条语句不等价?
(a) while (i < 10) {...}
(b) for(; i < 10; ) {...}
(c) do {...} while(i < 10);
答: 最后一条语句和其他两条语句不等价
